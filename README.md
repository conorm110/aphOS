# aphOS
aphOS is a simple 16 bit real mode DOS. Just for messing around with BIOS.

## Building and Running
**Building:** 
```
make
```
**Running:**
```
make run
```
Its not that deep.

## Requirements
- qemu
- nasm

**For Ubuntu**
```
sudo apt install nasm
sudo apt install qemu
sudo apt install qemu-system-i386
```
